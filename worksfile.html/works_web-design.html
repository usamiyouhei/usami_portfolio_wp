Web Design

  • Web Design

    自身のポートフォリオ

  • 制作過程

    普段、デザートやお菓子はアートになるので
    お洒落さやシズル感が重要視されます。
    デザインの視野で構築するのがとても勉強になりました。
    ロゴ製作のアイデアも引き出しがあるので、
    様々なことにチャレンジしていきたいです。

  • 目的

    Webサイト制作のデザインから学ぶことにより全体像を把握する。

  • 課題

    常に情報をアップデートし、良いデザインを学び続けたいです。
    職業柄、閃くことも多いので掛け算や引き算を意識していきます。

  • デザイン

    最初はグレーやグリーンのイメージで
    ポートフォリオのデザインを組んでいましたが、
    華やかなデザートの写真が多いので
    ピンクやオレンジ系で明るく仕上げました。
    様々なアニメーションも付けて、見ていて飽きないような
    工夫をしました。
